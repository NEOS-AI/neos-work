## Changes

### Task 1: Media FileViewer (PLAN Task 7)

- **`apps/server/src/lib/media-generator.ts`**
  - `listMediaFiles()` — `~/.neos-work/media` 스캔, kind/mime/size/createdAt
- **`apps/server/src/routes/media.ts`**
  - `GET /api/media/files?limit=`
- **`apps/desktop/src/pages/Media.tsx`**
  - 파일 목록 + 이미지/오디오 미리보기 (auth blob fetch)
- Sidebar/App 라우트 `media` + i18n

### Task 2: Deploy status polling (PLAN Task 8)

- **`apps/server/src/lib/deploy.ts`**
  - `getVercelDeploymentStatus`, `getCloudflareDeploymentStatus`
- **`POST /api/deploy/:id/refresh`** — 원격 상태 조회 후 DB 갱신
- **Deployments UI** — Refresh 버튼 + deploying/pending 15s 자동 폴링

### Task 3: Routine timezone / DST (PLAN Task 2)

- **`routine.timezone`** 컬럼 (기본 `UTC`)
- node-cron `timezone` 옵션으로 벽시계 스케줄 (DST 자동 반영)
- Routines UI timezone 프리셋 + 커스텀 IANA 입력

### Task 4: Coding block metadata (PLAN Task 12)

- `registerCodingBlocks()` 시 `WorkflowBlock` 메타 등록 → Blocks 목록/선택기에 노출
