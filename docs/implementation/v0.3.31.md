## Changes

### Task 1: Run duration timestamps (PLAN Task 14)

- **`formatDuration`** uses `parseTimestampMs` so SQLite `datetime('now')`
  run bounds compute correct durations
- Unit test for SQLite-style start/end strings

### Task 2: Media kind filter prefs (PLAN Task 7)

- **`loadMediaKindFilter` / `saveMediaKindFilter`** → `neos-media-kind`
- Media page restores last kind chip (all / image / audio / other)

### Task 3: Plugin pipeline + Skills Escape (PLAN Tasks 5 / 6)

- **PipelineRunner**: Escape closes modal and stops in-flight stream
- **Skills** try-prompt modal: Escape dismisses

### Task 4: Skills list polish

- Show **Installed {relative time}** when `installedAt` is present

### Task 5: Review fixes (carried from post-0.3.30 review)

- RevisionPanel: clear skip-blur flag when starting a new label edit
- `missing_edge_id` treats whitespace-only ids as missing
- WorkflowEditor: single Escape effect for Schedule + Shortcuts
- Engine tests: webhook endpoints + jsdom-safe export URL helpers
