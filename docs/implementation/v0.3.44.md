## Changes

### Task 1: MediaNode runtime hygiene (Task 7)

- Trim image `prompt` / audio `text` (reject whitespace-only)
- Clamp invalid image `size` → `1024x1024`
- Clamp invalid TTS `voice` → `alloy`
- Catch network errors instead of throwing

### Task 2: Parallel structure validation (Task 11)

- **`parallel_missing_start`**: Parallel End present without Parallel Start
  (symmetric to `parallel_missing_join`)

### Task 3: Slack content length

- **`SLACK_CONTENT_MAX_LENGTH`** (4000) in `media-node-options.ts`
- **`slack_content_too_long`** warning when any static body exceeds the limit

### Task 4: Escape UX

- **Templates**: Escape clears search (respects `defaultPrevented`)
- **ModeSelection**: Escape clears optional bearer token while not connecting

### Task 5: Unit tests

- MediaNode whitespace / size / voice / network cases
- WorkflowValidation parallel_missing_start + slack_content_too_long
