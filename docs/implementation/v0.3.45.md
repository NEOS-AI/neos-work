# v0.3.45 Implementation

## Summary

PLAN_FOR_V0_3_0 residual polish: **missing_llm_model** agent validation (Task 3),
**media quality** clamp + validation (Task 7), **whitespace block params**, and

## Changes

### Task 1: Agent model validation (Task 3)

- **`missing_llm_model`** warning when non-CLI agent has no `llmModel` / `model`
  (CLI providers skipped)

### Task 2: Media quality (Task 7)

- **`MEDIA_IMAGE_QUALITIES`**: `standard` | `hd`
- Graph: **`invalid_media_quality`** when set but not allowed
- **MediaNode**: invalid quality falls back to `standard`

### Task 3: Block param blank strings

- **`isBlank`** treats whitespace-only strings as empty for required `paramDefs`

### Task 4: Design Systems Escape (Task 1 UX)

- Escape while create form open → cancel create (unchanged)
- Escape while form closed and search non-empty → clear search
- Respects `defaultPrevented`

### Task 5: Unit tests

- missing_llm_model CLI vs model-set cases
- invalid_media_quality + whitespace block params
- MediaNode quality fallback / preserve hd

### Review fixes (post-ship)

- **AgentNode**: honor NodeConfigPanel `llmModel` (was only reading legacy `model`);
  prefer node `llmProvider` when building the LLM adapter
- **NodeConfigPanel**: explicit empty “Select model…” option so controlled value
  matches validation (`missing_llm_model`)
- **DesignSystems**: `cancelCreate` via `useCallback` for stable Escape deps
