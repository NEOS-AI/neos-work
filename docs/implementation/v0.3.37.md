# v0.3.37 Implementation

## Summary

PLAN_FOR_V0_3_0 residual UX polish: **Skills category filter prefs** (Task 5),
**Media Escape** clears preview, **Artifact Preview viewport prefs** (Task 4),
**missing_slack_content** validation, and **MCP add form** clears draft on dismiss.

기준: v0.3.36

---

## Changes

### Task 1: Skills category prefs (PLAN Task 5)

- **`skills-prefs.ts`**: `loadSkillsCategoryFilter` / `saveSkillsCategoryFilter`
- localStorage key: `neos-skills-category`
- Skills page restores last category chip; unknown categories fall back to `all`
  and rewrite prefs when the category disappears from the list

### Task 2: Media Escape (UX)

- Escape clears selected media file and blob preview URL

### Task 3: Artifact Preview viewport prefs (PLAN Task 4)

- **`artifact-preview-prefs.ts`**: `loadArtifactViewport` / `saveArtifactViewport`
- Shared modes: `ARTIFACT_VIEWPORT_MODES` (`full` | `tablet` | `mobile`)
- localStorage key: `neos-artifact-viewport`
- Preview toolbar restores last viewport mode

### Task 4: Graph validation

- **`missing_slack_content`** warning when Slack has channel but no text
  template/content and no upstream input

### Task 5: Settings MCP add form (UX)

- Cancel / Escape / successful create share `closeAddForm` which clears draft
  name/command/args/url and resets transport to stdio

### Task 6: Unit tests

- skills-prefs + artifact-preview-prefs round-trips
- ArtifactPreview viewport persist/restore
- WorkflowValidation missing_slack_content
