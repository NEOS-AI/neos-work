# v0.3.30 Implementation

## Summary

PLAN_FOR_V0_3_0 residual UX polish: **RevisionPanel Escape cancel** (Task 16 fix),
**relative timestamps** on Design Systems / Memory / Sessions, **Schedule/Shortcuts
Escape** close (Task 2/15 UX), **missing_edge_id** validation, and **exclude tests
from package builds** so `tsc` no longer typechecks vitest files.

기준: v0.3.29

---

## Changes

### Task 1: RevisionPanel Escape cancel (PLAN Task 16 fix)

- Escape while editing a label **cancels** edit without blur-saving
- `skipBlurSaveRef` prevents onBlur from calling `updateRevisionLabel`
- Empty label blur/Enter exits edit without writing
- Unit test: Escape while typing does not save

### Task 2: Relative times (list polish)

- **Design Systems**: Updated line uses `formatRelativeTime` + absolute `title`
- **Memory**: list timestamps relative; newest-first via `sortByDateDesc`
- **Sessions**: sort via `sortByDateDesc` (SQLite-safe); relative `updated_at` under title

### Task 3: Editor Escape affordances (PLAN Task 2 / UX)

- Escape closes **Schedule** routine modal
- Escape closes **Keyboard shortcuts** help
- Shortcuts help documents Esc for Schedule / Shortcuts

### Task 4: Graph validation

- **`missing_edge_id`** error when an edge has an empty id

### Task 5: Build hygiene

- Exclude `*.test.ts(x)` from `tsc` includes for desktop, server, and packages
  (core build no longer fails on orchestrator test casts)
