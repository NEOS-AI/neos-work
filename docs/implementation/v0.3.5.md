## Changes

### Task 1: Artifact refresh re-run (PLAN Task 4)

- **`POST /api/artifacts/:id/refresh`** body `{ mode: 'reload' | 'rerun' }`
  - `reload`: 기존 파일/DB 내용 재로드
  - `rerun`: `workflowId` 반환 → 클라이언트가 워크플로우 재실행
- **ArtifactPreview**
  - **Reload** / **Re-run** 버튼
  - `onRerunWorkflow` → WorkflowEditor `handleRun`

### Task 2: Revision snapshot dedup (PLAN Task 16)

- **`createRevision`**: 최신 snapshot과 동일하면 insert 생략 (`null` 반환)
- 저장 시 불필요한 history 노이즈 감소

### Task 3: Run log auto-scroll (PLAN Task 14)

- WorkflowEditor Run 탭: 새 이벤트/스트리밍 시 하단으로 `scrollIntoView`

### Task 4: Media delete

- **`DELETE /api/media/file/:filename`**
- Media FileViewer 삭제 버튼
- `EngineClient.deleteMediaFile`
