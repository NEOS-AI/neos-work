## Changes

### Task 1: Run History filter prefs (PLAN Task 14)

- **`run-history-filter.ts`**: `loadRunStatusFilter` / `saveRunStatusFilter`
- Shared chip list: `RUN_STATUS_FILTERS`
- localStorage key: `neos-run-history-status`
- RunHistoryPanel restores last status chip on mount

### Task 2: Run detail Escape (UX)

- **RunDetailPanel**: Escape calls `onClose` (same as ✕); stable listener via ref
- Detail panel hides when the active filter no longer includes the selected run

### Task 3: Design System editor dirty leave (PLAN Task 1)

- Back navigation confirms when DESIGN.md is dirty
- `beforeunload` warns on tab close / refresh while dirty

### Task 4: Graph validation (PLAN Task 11)

- **`gate_or`** treated like `or_gate` for underconnected OR race
- **`gate_and_underconnected`** when AND gate has fewer than two upstreams
- Parallel join detection includes `gate_or` / `gate_and`
- **`block_no_upstream`** warning for disconnected blocks in multi-node graphs

### Task 5: Unit tests

- run-history-filter prefs round-trip
- RunHistoryPanel restore/persist chip
- RunDetailPanel Escape
- WorkflowValidation gate_or / gate_and / block_no_upstream
