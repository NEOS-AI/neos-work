# v0.3.61 Implementation

## Summary

PLAN_FOR_V0_3_0 residual polish: **invalid JSON body → 400** on remaining
create/mutate routes, **domain/query normalize** (blocks list + harness domain),
and **session list workspaceId trim**.

기준: v0.3.60 (including review commit: blocks domain, media generate JSON,
Sessions optimistic message race)

---

## Changes

### Task 1: Invalid JSON create hygiene

Reject non-JSON bodies with 400 (instead of throwing) on:

- Workflow create
- Blocks create
- Memory create
- Routines create
- Harness create
- Session create / chat / agent
- Workspace create
- Media `/image` + `/audio`
- Deploy POST
- MCP server create

### Task 2: Query / domain normalize

- **Blocks list** `?domain=`: trim + lower-case
- **Harness create/PUT**: domain allow-list `finance|coding|general`
- **Session list** `?workspaceId=`: trim; blank → undefined (all)

### Task 3: Unit tests

- Invalid JSON create cases for workflow, blocks, memory, harness, routines,
  session, deploy, media, mcp
- Blocks domain clamp create/put
- Design Systems / workflow-revisions blank path id → 404
