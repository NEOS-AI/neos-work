## Changes

### Task 1: Coding domain size limits (Task 12)

- `code_eval`: reject code longer than 100_000 chars
- `file_read`: reject non-files and files larger than 2 MiB
- `file_write`: reject content longer than 2 MiB
- `safePath`: trim input before absolute/traversal checks

### Task 2: Store-layer id/key hygiene

- **Memory** `getMemory` (and dependents): trim id; blank → null
- **Settings** `getSetting` / `setSetting` / `deleteSetting`: trim key; blank no-op
- **Plugin** `resumeRun`: trim runId/stageId; blank → false

### Task 3: Design System create (Task 1)

- Trim name/description before charset validation and directory create

### Task 4: Live Artifact auto-save (Task 4)

- `createFirstHtmlArtifact`: require non-blank trimmed workflowId/runId; skip blank node ids

### Task 5: Unit tests

- Memory blank/trimmed id
- Settings key trim
- Plugin resume blank ids
- Design system name/description trim
- HTML artifact blank workflowId
- code_eval / file_write max size
