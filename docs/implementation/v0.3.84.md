# v0.3.84 Implementation

## Summary

PLAN_FOR_V0_3_0 residual polish: **test_runner go/cargo require `test`**
(Task 12), **coding/core path control-char reject**, **design-system/plugin
hidden-dir skip**, **WorkflowValidation cycle + edge-match trim** (Task 15),
**preflight edge endpoint trim**, **execution SERVER_URL http(s) only**, and
**MCP client listTools/callTool robustness** (Task 9).

기준: v0.3.83

## Changes

### Task 1: Coding blocks (Task 12)

- Shared `isAllowedTestCommand`: `npm|pnpm|yarn|pytest` any args; `go`/`cargo`
  require first arg `test` (case-insensitive basename)
- `safePath` / git_diff / test_runner reject `\0` / CR / LF
- Reject control chars in test_runner command and cwd

### Task 2: Design systems / plugins (Tasks 1/5)

- `listDesignSystems`: skip hidden directories (leading `.`)
- `listPlugins`: skip hidden skill directories

### Task 3: Core filesystem

- `safePath`: reject null bytes and CR/LF

### Task 4: WorkflowValidation (Task 15)

- Cycle detection uses trimmed node/edge endpoints; skip dangling edges
- Shared `endpoint` / `edgeTargetsNode` / `edgeSourcesFromNode` helpers
- All incoming/outgoing/isolated/upstream checks use trimmed id compare
- Self-loop + pair keys use `endpoint()` consistently

### Task 5: Preflight

- Node id set includes raw + trimmed ids
- Dangling edge check uses trimmed endpoints against that set

### Task 6: Execution settings

- `getExecutionSettings`: only inject runtime `SERVER_URL` when http(s);
  strip trailing slashes

### Task 7: MCP client (Task 9)

- `listTools`: tolerate non-array `tools`
- `callTool`: tolerate missing/non-array content; safe content item map

### Task 8: Unit tests

- go/cargo subcommand allow-list + control-char paths/commands
- buildNeosCliEnv non-http URL omit (coverage)
- hidden design-system / plugin dirs
- filesystem control-char path
- cycle with padded edge endpoints; gate underconnected with pad
- preflight padded/blank endpoints
- MCP missing tools/content
- media/deploy non-http SERVER_URL + error fallbacks
- shell / session / registry / web-search / path-safety coverage

### Task 9: Version

- monorepo **0.3.83 → 0.3.84**
- health / server banner / web_search User-Agent
