## Changes

### Task 1: Editor right-panel tab prefs (PLAN Tasks 4 / 14 / 16)

- **`layout-prefs.ts`**: `loadEditorRightPanelTab` / `saveEditorRightPanelTab`
- Shared tabs: `EDITOR_RIGHT_PANEL_TABS` (`config` | `run` | `history` | `preview`)
- localStorage key: `neos-editor-right-panel-tab`
- WorkflowEditor restores last right-panel tab on open; all tab switches persist
  (including auto-switch to run/preview/config during execution)

### Task 2: Memory type filter prefs

- **`memory-prefs.ts`**: `loadMemoryTypeFilter` / `saveMemoryTypeFilter`
- Shared chips: `MEMORY_TYPE_FILTERS` (`all` | `user` | `session` | `skill` | `reference`)
- localStorage key: `neos-memory-type`
- Memory page restores last type chip on mount

### Task 3: Design System editor Escape (PLAN Task 1)

- Escape navigates back to Design Systems list via `handleBack`
  (still confirms when DESIGN.md is dirty)

### Task 4: Graph validation

- **`multiple_triggers`** warning when more than one trigger node is present
- `no_trigger` still fires only when trigger count is zero

### Task 5: Unit tests

- layout-prefs right-panel tab round-trip
- memory-prefs type filter round-trip
- WorkflowValidation multiple_triggers
