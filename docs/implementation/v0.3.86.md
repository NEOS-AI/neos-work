## Changes

### Task 1: Memory tools

- Reject `\0` / CR / LF in remember/forget keys
- Drop control-char tags during normalize

### Task 2: Block registry / BlockNode (Task 12)

- `normalizeImplementationType`: native|prompt|skill; unknown → native
- Registry meta + BlockNode execution use shared normalizer
- Case-insensitive `NATIVE` / `PROMPT` / `skill`

### Task 3: Workflows DB

- Safe JSON parse for nodes/edges (non-array → `[]`) and nodeResults (non-object → `{}`)
- `updateWorkflow`: typeof guards; non-array nodes/edges → `[]`

### Task 4: Deploy helpers (Task 8)

- `safeDeployHostUrl`: bare host → `https://`; reject file:/javascript:
- Vercel/Cloudflare status URLs use the helper

### Task 5: Skill discovery

- Skip local scan when workspacePath has control characters

### Task 6: Runtime context

- `normalizeListenPort`: clamp 1–65535; invalid → fallback
- Trim auth token on set

### Task 7: Webhooks (Task 13)

- Cap body at 1 MiB (Content-Length + raw text)
- JSON arrays/primitives → empty trigger inputs

### Task 8: Browser tools

- Page API failures return `{ success: false }` instead of throwing

### Task 9: Discord / shell / plugin / media / rate-limit / HTML

- Discord non-ok responses include truncated body (500 chars)
- Shell: remove unused AbortSignal path; safe Error stringify
- Plugin list: whitespace-only SKILL.md → omit skillContent
- Media image download: http(s) only; cap 16 MiB
- Rate-limit: GC expired windows when map ≥ 256
- HTML artifacts: skip auto-save when content > 2 MiB

### Task 10: Unit tests

- memory key/tags control chars; discovery control-char path
- implementationType normalize + case; safeDeployHostUrl
- runtime port clamp; webhook 413; browser non-throwing
- Discord body truncate; plugin blank skill; HTML oversized; rate-limit GC
