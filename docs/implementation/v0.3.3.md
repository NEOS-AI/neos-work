## Changes

### Task 1: Claude Design ZIP import (PLAN Task 10)

- **`apps/server/src/routes/workflow.ts`**
  - `POST /api/workflow/import.zip` — `workflow.json` 없으면 HTML entry(`index.html` 등) 감지 → Claude Design 경로
  - 워크플로우(trigger→output) 생성 + HTML artifact 저장
  - `POST /api/workflow/import/claude-design` — 전용 엔드포인트
  - 이름 충돌 시 `Copy of …` suffix

- **Desktop**
  - `importClaudeDesignZip()` + Workflows **Import (Claude Design)** 버튼
  - 일반 ZIP import도 HTML-only 자동 폴백

### Task 2: Richer ZIP export (PLAN Task 10)

- Export ZIP에 포함:
  - `workflow.json` (+ designSystemId, counts)
  - `README.md`
  - `runs/<runId>.json` (최근 10개)
  - `artifacts/*.{html,md,txt}` + `.meta.json`
- Import 시 `artifacts/` 파일 재생성

### Task 3: Agent streaming Run Log (PLAN Task 14)

- **WorkflowEditor** Run 탭
  - 동일 node의 연속 `node.progress` 이벤트 하나로 collapse
  - 스트리밍 행 표시 + 누적 텍스트 expand

### Task 4: Skill → Plugin upgrade (PLAN Task 5)

- **`plugin-store.upgradeSkillToPlugin()`** — `open-design.json` 4-step pipeline 생성
- **`POST /api/plugins/upgrade-from-skill`**
- Skills UI **→ Plugin** 버튼

### Task 5: Coding blocks registration (PLAN Task 12)

- 서버 기동 시 `registerCodingBlocks()` 호출 (finance와 함께)
