## Changes

### Task 1: DeployNode network + settings hygiene (Task 8)

- Wrap `fetch` in **try/catch** (network errors → `ok: false`, not throw)
- Trim `SERVER_URL` / `SERVER_TOKEN` before the deploy API call

### Task 2: MediaNode settings hygiene (Task 7)

- Trim `SERVER_URL` / `SERVER_TOKEN` for image and audio generation requests

### Task 3: AgentNode API key / memory URL hygiene (Task 3)

- **buildAdapter**: trim OpenAI/Anthropic/Google keys and base URLs;
  whitespace-only Anthropic/Google keys treated as missing
- **Google** with empty key errors explicitly (`GOOGLE_API_KEY is not configured`)
  instead of falling through to Anthropic
- Trim `SERVER_URL` / `AUTH_TOKEN` used for memory export injection

### Task 4: Unit tests

- Deploy network failure + URL/token trim
- Media URL/token trim
- Agent whitespace Anthropic key + empty Google key
