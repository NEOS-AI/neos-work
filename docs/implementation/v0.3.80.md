## Changes

### Task 1: Skill parser + discovery

- `parseSkillFile`: reject blank content; trim name (blank → null); lower-case
  mode/category; trim version/license/path/body
- `discoverSkills`: trim workspacePath (blank → no local scan); skip hidden
  `.md` files; trim scan directory path

### Task 2: Context manager

- Clamp threshold 1–2M (invalid → default 80k)
- Empty/non-array messages → no compression
- Non-array compress → `[]`

### Task 3: Crypto

- `encrypt`: coerce non-string to string
- `decrypt`: trim; reject non-encrypted / blank with clear error
- `isEncrypted` allows empty ciphertext body (empty plaintext round-trip)

### Task 4: Unit tests

- parser blank content + mode/category case
- discovery blank workspace + hidden file skip
- context-manager empty messages + threshold clamp
- crypto invalid decrypt + non-string encrypt
