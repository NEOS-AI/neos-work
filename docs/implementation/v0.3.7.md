## Changes

### Task 1: Revision restore dirty warning + counts (PLAN Task 16)

- **`RevisionPanel`**
  - dirty 시 restore confirm (unsaved 폐기 경고)
  - clean 시에도 복원 confirm
  - 목록에 node/edge count 표시
- **`listRevisions`**
  - snapshot에서 `nodeCount` / `edgeCount` 파생 (raw snapshot 미반환)
- WorkflowEditor → `isDirty` prop 전달

### Task 2: Artifact content-type preview (PLAN Task 4)

- **`ArtifactPreview`**
  - HTML → sandboxed iframe + Full/Tablet/Mobile viewport
  - Markdown / plain text → monospaced renderer (iframe 아님)
  - **Delete** 버튼 → `DELETE /api/artifacts/:id`
- **`EngineClient.deleteArtifact`**

### Task 3: Parallel / OR / media / deploy validation (PLAN Task 11, 7, 8)

- **`validateWorkflowDraft`**
  - `parallel_start` fan-out < 2, `parallel_end` / `or_gate` join < 2 경고
  - `parallel_start` without join 경고
  - media prompt/upstream 경고, deploy provider 필수

### Task 4: Webhook UX (PLAN Task 13)

- Config Webhook 섹션: **Copy URL**, **Copy curl** (HMAC 예시), secret **Copy**

### Task 5: CLI stderr progress (PLAN Task 3 / 14)

- `spawnCliAgent` stderr chunk도 `onChunk`로 스트리밍
