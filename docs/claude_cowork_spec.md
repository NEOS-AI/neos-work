# [사양 정의서] Claude Cowork

## 1. 개요 (Overview)
*   **서비스 명칭:** Claude Cowork (클로드 코워크)
*   **정의:** 사용자가 설정한 목표(Outcome)를 달성하기 위해 스스로 계획을 세우고, 로컬 파일 시스템 및 웹 브라우저를 통해 다단계 작업을 자율적으로 수행하는 AI 에이전트 서비스.
*   **핵심 가치:** "Prompt to Done" – 단순히 답을 주는 것이 아니라, 실제 결과물(파일 정리, 보고서 작성, 데이터 추출 등)을 만들어내는 실행 중심의 도구.

## 2. 주요 기능 및 특징 (Key Features)
### 2.1 에이전틱 작업 실행 (Agentic Task Execution)
*   사용자가 결과(Outcome)를 설명하면, Claude가 이를 달성하기 위한 하위 단계(Sub-tasks)를 스스로 설계하고 실행합니다.
*   실행 중 발생하는 오류를 스스로 감지하고 수정(Self-healing)하며 작업을 완수합니다.

### 2.2 로컬 폴더 연동 (File-based Workflow)
*   사용자가 지정한 로컬 폴더에 직접 접근하여 파일을 읽고, 수정하고, 생성합니다.
*   **지원 작업:** 수백 개의 파일 분류, 영수증 폴더를 읽어 지출 보고서(Excel) 작성, 대량의 파일 이름 일괄 변경 등.

### 2.3 확장된 도구 사용 및 스킬 (Plugins & Skills)
*   **Native Skills:** Microsoft PowerPoint(.pptx), Word(.docx) 파일을 직접 생성하거나 편집하는 전용 스킬 제공.
*   **MCP(Model Context Protocol) 지원:** 외부 데이터 소스나 도구와 연결할 수 있는 개방형 프로토콜을 통해 기능을 확장.
*   **브라우저 자동화:** 필요 시 웹 브라우저를 사용하여 정보를 검색하거나 데이터를 수집.

## 3. 기술 사양 (Technical Specifications)
*   **실행 환경:** Claude Desktop 앱 (현재 macOS 전용)
*   **대상 요금제:** 유료 플랜 사용자 전용 (Claude Pro, Max, Team, Enterprise)
*   **구동 방식:** Anthropic의 에이전틱 아키텍처(Claude Code와 유사한 엔진) 활용.
*   **인터페이스:** 채팅 창 하단의 전용 'Cowork' 모드 활성화.

## 4. 보안 및 데이터 정책 (Security & Privacy)
*   **로컬 저장:** 대화 내역 및 작업 로그가 사용자 컴퓨터에 로컬로 저장됩니다 (Anthropic 서버에 영구 보관되지 않음).
*   **권한 제어:** 사용자가 명시적으로 허용한 폴더에 대해서만 접근 권한을 가집니다.
*   **네트워크 보안:** 사용자의 현재 네트워크 정책 및 이그레스(Egress) 권한을 준수합니다.
*   **제한 사항:** 현재 리서치 프리뷰 단계로, 감사를 위한 오딧 로그(Audit Logs)나 규제 대상 워크로드(Regulated Workloads)에는 권장되지 않습니다.

## 5. 주요 사용 사례 (Use Cases)
| 카테고리 | 상세 내용 |
| :--- | :--- |
| **파일 정리** | "다운로드 폴더의 파일을 날짜와 유형별로 정리해줘" |
| **데이터 처리** | "폴더 내 모든 PDF에서 주요 수치를 추출해 엑셀로 요약해줘" |
| **문서 자동화** | "조사한 내용을 바탕으로 10페이지 분량의 PPT 발표 자료를 만들어줘" |
| **미디어 관리** | "이미지 파일들을 웹용으로 압축하고 일관된 이름으로 변경해줘" |

## 6. 현재 제한 사항 (Current Limitations)
*   **플랫폼 제한:** 현재 macOS에서만 사용 가능하며, 기기간 동기화가 지원되지 않습니다.
*   **세션 유지:** 데스크톱 앱이 실행 중인 상태에서만 작업이 지속됩니다 (앱 종료 시 세션 종료).
*   **프로젝트 미지원:** 현재 Claude의 'Project' 기능과는 별도로 작동합니다.
*   **메모리:** 세션 간의 연속적인 메모리 보존이 아직 완벽하지 않은 리서치 프리뷰 상태입니다.
